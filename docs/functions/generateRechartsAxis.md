## [generateRechartsAxis](src/functions/generateRechartsAxis.ts)
A function to generate data for an axis in a chart created with [recharts](https://recharts.org/).

If the amount of data ticks you have is too large to display labels for all of them on an axis, recharts will cut them down to intervals automatically.
This works great for certain data, but sometimes intervals of 5 or 10 are more pleasing than intervals of 7, and a label every hour is more useful than a label every 70 minutes.

This helper allows you to access the specific implementation of [RechartsAxis](docs/components/RechartsAxis.md).
Use this helper if you do not want a full implementation but would rather just use a certain part of the data generated by that helper.

| Parameter | Description | Type | Required/Default |
|------|-------------|------|------------------|
| options.start | The lowest value on your axis | number/T | required |
| options.end | The highest value on your axis | number/T | required |
| options.divisor | The parameter you're using this component for. All intervals are multiples of this property. If you supply an array of divisors, the component will first try to use other divisors before considering multiples of the last one | number/Array<number> | required
| options.maxTicks | The maximum amount of labels you have space for | number | required |
| options.minTicks | The minimum amount of labels you want to have | number | required |
| options.tickFormatter | A function to calculate the your tick values after calculation of tick number and interval length | (start: number, intervalLength: number, i: number) => number | (start, length, i) => (start + length * i) |
| **@returns** | A tickInfo object with the keys described below | | |

### TickInfo

| Key | Description | Type |
|-----|-------------|------|
| intervalCount | The number of ticks | number |
| intervalLength | The length of the interval between each tick | number |
| min | minimum value | number/any |
| max | maximum value | number/any |
| ticks | all ticks | Array<number/T> |

### useRechartsAxis
This function is exported as a memoized hook called `useRechartsAxis(options, deps)`. 

### Examples

```jsx
import { generateRechartsAxis } from './generateRechartsAxis';

const tickInfo = generateRechartsAxis(
    0,
    100,
    5,
    6,
    3
);

return <XAxis
    ticks={tickInfo.ticks}
    tickCount={tickInfo.intervalCount}
    domain={[tickInfo.min, tickInfo.max]}
    interval="preserveStart"
/>
```
